# budsBags

A high-performance Bagnon-style inventory Addon for World of Warcraft (3.3.5 / Ascension) with built-in ArkInventory-style visually separated auto-categorization and ElvUI-style sorting.

## Features

- **Bagnon-Style View**: All of your bags and your backpack are seamlessly merged into one large, clean window.
- **Auto-Categorization**: Items are automatically separated into groups like Gear, Consumables, Trade Goods, Quest Items, and Trash for maximum clarity.
- **Fast Sorting Engine**: Extremely performant sorting algorithm that quickly visually reorders the items in your bags.
- **Item Level Overlay**: Clean overlay of the item's level on top of the button icon, perfect for min-maxing and easily parsing Ascension items.

## Installation

1. Download the latest release from the repository.
2. Extract the folder into your World of Warcraft AddOns directory:
   `World of Warcraft\Interface\AddOns\budsBags`
3. Log into the game and ensure the AddOn is enabled.
